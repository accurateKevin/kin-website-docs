public enum Stellar

    Declaration

    Swift

    public static func transaction(source: Account,
                               destination: String,
                               amount: Int64,
                               memo: Memo = .MEMO_NONE,
                               node: Node,
                               fee: Stroop) -> Promise<TransactionEnvelope>

    Parameters

    source

    The account from which the payment will be made.
    destination

    The public key of the receiving account, as a base32 string.
    amount

    The amount to be sent.
    memo

    A short string placed in the MEMO field of the transaction.
    node

    An object describing the network endpoint.
    fee

    The fee in Stroops used when the transaction is not whitelisted.

    Return Value

    A promise which will be signalled with the result of the operation.

  • txWatch(account:lastEventId:node:)

    Observe transactions on the given node. When account is non-nil, observations are limited to transactions involving the given account.

    Declaration

    Swift

    public static func txWatch(account: String? = nil,
                           lastEventId: String?,
                           node: Node) -> EventWatcher<TxEvent>

    Parameters

    account

    The Account whose transactions will be observed. Optional.
    lastEventId

    If non-nil, only transactions with a later event Id will be observed. The string now will only observe transactions completed after observation begins.
    node

    An object describing the network endpoint.

    Return Value

    An instance of TxWatch, which contains an Observable which emits TxInfo objects.
